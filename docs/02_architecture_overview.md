## ユーザーフロー (4ステップ)

1. **写真撮影** - スマートフォンから全身写真をアップロード
2. **身体測定** - システムが自動で体形を検出し寸法を計算
3. **衣服選択** - カテゴリ別（トップス、ボトムス等）に閲覧
4. **試着画像生成** - AIが生成したリアルな着用画像を表示

## コア技術レイヤー

### Layer 1: Body Understanding (身体認識)
- 骨格18点検出
- 肩幅・袖丈・股下などの実寸法計算
- テスト被験者10名での検証済み

### Layer 2: Garment Fitting (衣服フィッティング)
- ディープラーニングによる衣服変形
- 平面商品画像 → 体型・ポーズに合わせた変形
- 単なる合成ではなく「着ている」ように見える処理

### Layer 3: Image Composition (画像合成)
- フィットした衣服と元画像のシームレス融合
- 肌と生地の境界処理
- テクスチャ保持
- 体型比率の維持
