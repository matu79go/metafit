# 技術ブログ: AI仮想試着の進化 - PASTA-GAN++からGemini Nano Bananaへ

> このドキュメントは開発の試行錯誤を記録し、最終的に技術ブログ記事としてまとめるためのものです。

## はじめに

オンラインショッピングにおける最大の課題の一つ、「試着できない」問題。返品率30-40%という数字が示すように、消費者は購入前に「自分にどう見えるか」を確認したいというニーズを持っています。

本プロジェクトでは、AIを活用したバーチャル試着システムの開発に取り組んでいます。

## Chapter 1: 従来アプローチとその限界

### PF-AFN による2Dワーピング
最初に試みたのは PF-AFN (Pose Flow Attention Flow Network) による2D衣服ワーピングです。

**仕組み:**
1. 衣服画像を入力
2. ターゲット人物のポーズを検出
3. AFWM (Attention Flow Warp Module) で衣服をポーズに合わせて変形
4. ResUnetGenerator でブレンディング・精製

**結果:** 基本的な試着は実現したが、以下の問題が残りました:
- 複雑なポーズでの歪み
- 体型の多様性への対応不足
- 衣服のディテール（ロゴ、パターン）の崩れ

### PIFu による3Dアプローチ
次に PIFu (Pixel-aligned Implicit Function) で3D復元を試みました。

**結果:** 3D復元自体は成功したが、試着システムへの統合が複雑で、リアルタイム性に欠けました。

### PASTA-GAN++ による統合アプローチ
最終的に PASTA-GAN++ をベースとしたシステムを構築しました。

**改善点:**
- OpenPose による精密なポーズ検出（18点）
- Graphonomy による人体セグメンテーション
- 上半身 / 下半身 / 全身の個別対応

**残った課題:**
- 処理速度（1枚あたりの生成時間が長い）
- 非標準体型での品質劣化
- タイトな衣服での再現性
